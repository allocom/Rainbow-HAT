Welcome
-------

This documentation will guide you through the methods available in the Rainbow HAT python library.

Rainbow HAT is a Raspberry Pi add-on with 32/64 individually controllable RGB LEDs.


* GPIO pinout - https://pinout.xyz
* Get the code - https://github.com/allocom/rainbow-hat
